# 📊 Superstore Sales EDA & Dashboard

## Overview
Analyzed 10,000+ retail transactions to uncover profit trends,
seasonal patterns, and the impact of discounting on sales.

## Key Insights
- Technology is the top revenue category ($836,154)
- West region generates the highest profit ($108,418)
- 18.7% of orders are loss-making
- 1,113 orders with high discounts run at a loss
- November 2017 was the best sales month

## Tech Stack
Python | Pandas | Matplotlib | Seaborn | JupyterLab

## Files
- `analysis.ipynb` — Full EDA with 6 visualizations
- `Sample - Superstore.csv` — Dataset
